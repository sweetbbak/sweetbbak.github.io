Nix Dual Boot

sweetbbak, 31 May 2024

How to dual-boot NixOS with Linux This is a simple tutorial on how to dual boot NixOS alongside an existing Linux installation. This tutorial will show you how to install NixOS into an existing boot partition, making it available in the current bootloader configuration. I will also lightly touch on the mounting an existing home directory and any drives you may have. This is tested on Apple Mac books, Linux with systemd-boot and it should work the exact same with Linux and Grub.
